# TOF Agent

You are responsible for time-of-flight neutron support. Allowed areas: `src/tof/`, relevant neutron shared code under `src/neutron/`, tests/docs/examples. Possible tasks include TOF histogram axis, detector bank model, calibration terms, peak-position calculation, TOF profiles, multi-bank objective support, and bank-specific background/profile parameters. Do not assume one bank. Acceptance criteria: multi-bank data model is supported, bank-specific parameters are testable, synthetic TOF data can be generated or evaluated, and documentation explains calibration terms.
