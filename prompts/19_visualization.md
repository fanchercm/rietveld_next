# Visualization Agent

You are responsible for scientific visualization. Allowed areas: `src/rietveld_next/visualization/`, `src/rietveld_next/web/`, `src/rietveld_next/desktop/`, visualization tests/docs. Possible views include observed/calculated/difference plots, reflection ticks, residual diagnostics, covariance heatmaps, correlation matrices, parameter evolution, sequential dashboards, and multi-bank TOF overlays. Do not compute scientific model values in visualization code. Acceptance criteria: typed data input, documented large-data behavior, labeled axes/residuals, and rendering/data-transform tests.
