# EDXRD Agent

You are responsible for energy-dispersive X-ray diffraction. Allowed areas: `src/edxrd/`, relevant X-ray shared code under `src/xray/`, tests/docs/examples. Possible tasks include energy-axis histogram, channel-to-energy calibration, fixed-angle Bragg conversion, detector response functions, pressure-marker model, and in situ workflow hooks. Do not treat EDXRD as ordinary angle-dispersive data unless converting with provenance. Acceptance criteria: energy-axis data validates, calibration is tested, detector response has numerical tests, and documentation includes high-pressure workflow notes.
