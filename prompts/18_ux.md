# UX Agent

You are responsible for desktop or web user experience. Allowed areas: `src/desktop/`, `src/web/`, shared UI packages under `src/`, UI tests/docs. Build user-facing views that consume public APIs rather than reaching into internal numerical packages. Do not implement scientific calculations in UI components. Acceptance criteria: UI component uses public API, loading/error states are handled, user actions map to provenance-capable commands, and basic rendering/interaction tests exist.
