# Rietveld Public Backend Corpus

This package contains public GSAS-II tutorial files for MS01 backend testing.

Included capabilities:

- CIF import smoke tests: CuO, CuCr2O4, NAC, CaF2
- CW instrument parameter parsing: BT1.prm, 11bm_gsas.prm
- Synchrotron CW XRD profile import: 11BM_NAC.fxye
- TOF neutron profile import: PG3_22048.gsa, PG3_22049.gsa
- GSAS-II TOF instrument parameter parsing: POWGEN_1066.instprm, POWGEN_2665.instprm
- Legacy GSAS project/reference parsing: NAC-2015A.EXP

The source pages are public GSAS-II tutorial data pages:

- https://advancedphotonsource.github.io/GSAS-II-tutorials/Simulation/data/
- https://advancedphotonsource.github.io/GSAS-II-tutorials/TOF-CW%20Joint%20Refinement/data/

See `manifests/public_backend_corpus.yaml`, `manifests/file_inventory.tsv`, and `manifests/SHA256SUMS.txt`.
