# License And Citation Guide

## Purpose

This guide records licensing and citation expectations until formal release
metadata is finalized.

## Scope

Contributors must preserve third-party notices, cite scientific sources used for
reference data or formulas, and avoid committing non-redistributable datasets.

## Non-Goals

This guide does not assign a final project license. A formal license file should
be added before the first public release.

## Citation Guidance

Scientific docs and validation fixtures should cite source datasets, formulas,
and external software comparisons. When no validation source exists, record that
limitation in the validation report.
