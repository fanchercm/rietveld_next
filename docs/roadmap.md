# Roadmap

## Purpose

This roadmap summarizes near-term development direction for contributors and
agents.

## Scope

The project advances from foundation metadata and validation toward complete
physics, optimization, workflows, UX, AI, HPC, and benchmark capabilities.

## Non-Goals

This roadmap is not a promise of release dates or scientific validation status.

## Milestone Themes

- M01-M03: source layout, project model, storage foundation.
- M04-M11: objectives, derivatives, profiles, optimizers, diagnostics.
- M12-M28: structural IO and X-ray, neutron, TOF, magnetic, EDXRD physics.
- M29-M33: workflows, AI, UX, visualization, HPC.
- M34-M35: validation, documentation, governance.
- M36-M40: benchmark infrastructure and benchmark suites.
